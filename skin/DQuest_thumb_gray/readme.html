-------------------------------------------------------------
DQThumb Skin 1.07 제작:드림퀘스트 http://www.dqstyle.com
-------------------------------------------------------------

"DQThumb Skin"은 사진 갤러리를 위한 스킨입니다.
*사진갤러리 이외의 용도에는 맞지 않을수 있습니다.
*Gray Color버전과 White Color버전이 합께 압축되어 있습니다.


■ 기능설명
-섬네일 형식으로 리스트를 보여줍니다.
-섬네일을 보여줄때 GD를 이용하여 대칭리사이즈후 서버에 저장한뒤 브라우저에 뿌려주므로 속도가 빠르고 트래픽 문제가 없습니다.
-섬네일 별도생성이므로 파일을 두개까지 업로드 가능합니다.
-이미 생성한 섬네일은 중복 생성하지 않아 속도가 빠릅니다.
-섬네일의 배치를 상하좌우 어느쪽이던 이동시킬수 있으며 그에 맞게 게시물의 정보가 자동으로 정렬됩니다. (setup.php 설정필요)

-업로드된 사진에 exif정보가 있다면 촬영시의 조건이 그대로 출력됩니다.

-사진을 업로드 하지않고 HTML태그만으로 구성된 게시물의 경우 태그를 분석하여 섬네일을 추출/생성해냅니다.
-제로보드의 그림창고(ImageBox) 기능으로 연작 사진을 업로드할 경우 생성되는 전용태그를 img태그로 번역하는 기능이 있어서 한번 게시한 게시물을 다시 타 사이트에 링크시 아주 유용합니다.
-일정시간이 지나면 자신의 글에 추천할수 있게되는 현상을 막음(중복투표를 막는것과는 다름)
-사진감상에 방해가 되지 않도록 게시물에 대한 정보는 모두 아래쪽에 나옵니다.(글제목은 예외)
-촬영장소와 사진설명을 입력할수 있습니다.(Link#1과 Link#2를 사용 가능하도록 설정하여야 함)
-사진감상에 편리하도록 사진을 가운데로 정렬해서 보여줍니다.
-게시판의 크기보다 큰 사진을 올렸을때 폼이 깨지는것을 방지하기위해 사진이 출력되는 영역의 크기가 유동적으로 변하도록 하였습니다.
-사진이 출력되는 영역의 폭을 별도 지정가능합니다.


■ 설치시 주의점
ftp 프로그램으로 업로드한후 exiflist(확장자 없음)파일의 퍼미션을 변경해 주어야 합니다.
권장 퍼미션은 505 이상입니다.
퍼미션을 변경해주지 않으면 exif정보가 나오지 않습니다.
주!)setup.php 파일을 열어서 용도에 맞게 설정을 바꾸어주셔야 합니다.
섬네일 이미지의 출력이 이상할 경우 반드시 setup.php 파일의 설정을 변경하셔야 합니다.
설정 방법은 setup.php 내부에 주석으로 설명되어 있습니다.

■ $_gd_on=false; 옵션의 사용 (GD기능을 사용하지 않음)
setup.php 파일에 준비되어 있는 이 옵션은 GD가 설치되어있지 않은 서버에서 본 스킨을 사용하기 위해 준비된 옵션입니다.
이 옵션을 사용했을때는 make_thumb.php 파일 대신, make_thumb_nogd.php 파일을 인클루드하는 구조로 되어 있으며 기능상 많은 제약이 따르게 됩니다. 우선, gd 사용시의 장점을 살리지 못하게 되기 때문에 트래픽이 상당하고 속도도 느립니다. 결국 원본과 비율이 같은 대칭리사이즈가 가능하다는 장점 이외엔 섬네일에 대한 추가기능은 없어지게 됩니다.
이 옵션을 사용함으로 해서 생기는 제한 사항은 다음과 같습니다.
-GD의 장점을 살리지 못해서 트래픽 증가와 속도저하 현상이 생긴다.
-HTML태그 분석으로 섬네일을 출력할때 원본크기의 이미지가 서버에 다운로드 되어 누적된다.
-테스트 부족으로 예상치못한 버그가 있을수 있음

■ 기타
::필요없는 섬네일 파일의 정리
제로보드의 관리자모드에서 '첨부파일 정리'를 들어가신 후 '쓰레기파일 정리'를 누르시면 섬네일 파일과 폴더들이 모두 제거됩니다.

*서버에 설치된 GD의 버전에 따라 일부함수가 정상동작하지 않을수 있습니다(에러발생 원인).
*new21,비누넷,SSHEL,cafe24 호스팅과 APM_SETUP4에서 정상동작 하는것을 확인하였습니다.

*exif정보는 리눅스서버에서만 나옵니다.(APM, iis에서는 출력되지 않음)
 *gif 파일은 섬네일이 생성되지 않습니다. 되도록이면 jpg,png 파일만 업로드하도록 해주십시오.

*관리자 설정에서 "글내용볼때 아래에 전체 리스트 기능"을 켜게되면 게시물 보기에서 '이전사진, 다음사진'이 나오지 않습니다.

※임의의 기능추가와 외형변경 허용됩니다. 허락받지 않으셔도 됩니다.
단, 예쁘게 변경하신 스킨은 제 홈페이지나 nzeo에 등록해 주시면 고맙겠습니다.(강제성 없음)

※이메일로는 질문을 받지 않습니다. 제 홈페이지의 스킨관련 게시판에 질문해 주시면 가장 빠르게 답변해 드립니다.


■ 버전별 변경내용
- 1.07
GD 2.0.1 이상이 설치된 경우 섬네일 이미지의 퀄리티가 훨씬 좋아졌습니다.(업그레이드시에는 기존 섬네일 삭제요망)
섬네일 사이즈를 이전보다 큰 사이즈로 변경하면 섬네일이 자동 재생성 되지않던 문제 해결
짧은답글(Comment) 입력 상자의 디자인 변경(로그인 하지 않았을때 차이 납니다.)

- 1.06b
PHP 4.3.2 버전이 설치된 서버에서 동작시 HTML 태그를 이용한 링크이미지의 섬네일 생성에서 발생하던 문제 해결 (NEW21, 비누넷, Cafe24....)

- 1.06a
$_auto_resize=true; 옵션 사용시 테이블이 깨지는 현상을 고쳤습니다.
exif 추출시 field.txt 파일을 사용하지 않도록 하여 ascii 모드로 업로드하지 않아도 정상적으로 출력되도록 하였습니다.

- 1.06
GD라이브러리가 설치되어 있는지 스킨 자체에서 판단하여 환경설정을 자동 조절합니다.(수동설정 가능)
imageCreatetruecolor()함수의 존재 유무를 스킨에서 판단하여 환경설정을 자동 조절합니다.
(일부 서버에서 오동작 가능성(예:비누넷)이 있으므로 GD관련 에러가 발생하면 수동설정이 필요할수도 있음
"setup.php"파일에 $_auto_resize 변수를 추가하여 사진의 가로폭이 게시판의 가로폭보다 클때 자동리사이즈 하는 기능을 끄고 켤수 있게끔 하였습니다.

- 1.05
섬네일의 수직정렬을 "middle"로 했을때 섬네일 아래의 글이 벌어지지 않고 섬네일 바로밑에 오던 문제점 고침
GIF파일의 링크시 섬네일 생성에 생기던 문제점 고침
파일을 업로드 하지않고 타 사이트에서 링크한 게시물에서도 img태그를 분석, 섬네일을 생성하도록 기능추가
링크에서 섬네일 생성시 깨어진 링크 자동판단기능 추가
그림창고로 올려진 그림들을 타 사이트에 링크할수 있는 img태그로 소스 번역하는 기능 추가
(게시물 작성후 다시 수정모드로 들어가면 번역되어 나옵니다.)

- 1.04
make_thumb.php 파일의 오타로 인해 $gd2=false; 옵션 사용시 문제가 생기던것을 고침

- 1.03
GD라이브러리가 없더라도 사용할수 있는 옵션추가
exif출력기능을 켜고 끌수 있게하는 옵션추가
jpg파일이 아니더라도 모든 종류의 이미지가 대칭리사이즈 되도록 기능추가
png파일 지원기능 추가
섬네일 파일의 크기 변경시 기존 파일을 수동으로 지우지 않아도 새로운 파일이 생성 되도록 수정
섬네일의 정렬방식을 마음대로 선택할수 있는 옵션추가
게시물보기에서는 카테고리 선택상자 안나오게 수정(제로보드의 구조상 동작불가)
게시물보기에서 사진정보와 함께 '이전사진'과 '다음사진'의 링크가 섬네일로 나오는 기능 추가

- 1.02
로그인 해야만 카테고리 선택상자가 보이던 문제해결
리스트보기에서 각 게시물마다 카테고리가 표시되는 기능추가
게시물보기에서 카테고리가 '분류'로 표시되어 나오는 기능추가
게시물보기에서 '이전사진'과 '다음사진'으로 넘어갈수 있는 기능추가

로그인후 추천기능을 사용할 수 없게 되는 문제해결
